For HTTPS (Recommended for new users):

You likely have cached credentials for a different GitHub account.

Windows: Open the Credential Manager (search for "Credential Manager" in the Start menu), select Windows Credentials, find the generic credentials related to github.com, and remove them. The next time you git push, you will be prompted to sign in with your authorized GitHub account.

macOS: Open Keychain Access (search for "Keychain Access" in Spotlight), search for "GitHub", and delete the relevant entry.
